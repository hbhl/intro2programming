What is our objective?

In partnership with Healthy Brains for Healthy Lives and McGill's Skillsets program, this workshop aims to educate scientists in the basics of programming so that they can begin to apply these skills to their independent research. With an initial introduction to basic programming structure and an overview of languages and tools of interest, there will also be an emphasis on developing reproducible and portable software.

The layout of this event is in three half-day components, each of which may be attended independently, and covering the following content:

  • Session 1, Introduction to Programming: this session is intended for true beginners. We will cover the basics of programming, how algorithms are designed, and begin practicing with our own non-research examples. We will budget some time in for helping configure participants' computers for the later sessions.

  • Session 2, Fundamentals of Programming: this session will jump through components of the Software Carpentry curriculum, with particular emphasis on programming in Python as it is the most syntactically accessible for beginners, and the structure introduced can be applied in R, as well.

  • Session 3, Applications in Data Analysis: this session will in fact be two sessions run in parallel, one for Python and the other for R. For Python, we will be introducing the Scikit-learn library for exploring and visualizing data and performaing basic statistical processing and analysis. For R, we will be following closely the Data Carpentry in R lesson which analyzes ecological data, but can be easily generalized to neuroscience data.

The organizers hope the outcome of this event is an increased understanding of programming methodologies for data analysis, and, in particular, increased ability of users to know what questions to ask in and where to ask them in order to expedite and ensure high quality in their research.
